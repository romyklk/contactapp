## Installation
1. Clonez le projet
2.  `composer install`
3.  `php bin/console doctrine:database:create`
4.  `php bin/console doctrine:migrations:migrate`
5.  `php bin/console doctrine:fixtures:load`
6.  `symfony server:start -d` 
7.  `symfony open:local`
